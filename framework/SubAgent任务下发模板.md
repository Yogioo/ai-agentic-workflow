# SubAgent 任务下发模板规范

## 1. 文档定位

本文档定义 Main Agent 向 Dev Sub Agent 与 QA Sub Agent 下发任务时必须遵循的统一格式。

若 Sub Agent 已通过系统提示词、agent 配置或运行平台内置机制获得对应角色规范，则任务包中不得再次要求其读取 `DevSubAgent.md`、`QASubAgent.md` 或其他角色定义文档。

角色规范文档用于 agent 定义与初始化，不作为每次派单的重复必读输入。

每次派单应优先提供：

- 任务目标
- 接口契约
- 验收标准
- 文件路径范围
- 本次任务相关事实文档
- 回传要求
