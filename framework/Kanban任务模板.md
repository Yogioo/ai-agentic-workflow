# Kanban 任务模板规范

## 1. 文档定位

本文档定义 Main Agent 在维护“当前阶段 Kanban”时应使用的统一结构与填写规则。

Kanban 的职责不是替代 PRD、技术方案或 RoadMap，而是将“当前阶段真正要执行的任务”整理为可调度、可验收、可追踪的任务清单。

---

## 2. 使用范围

- 一个 Kanban 文档只服务当前阶段或当前迭代。
- Kanban 只应细化当前阶段，不对未来阶段过度展开。
- Kanban 是任务目录，不直接替代正式任务包。

---
